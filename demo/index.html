What is kairu?

Kairu is a small, dependency-light inference optimizer for autoregressive LLMs. It implements the same techniques the big serving frameworks use — speculative decoding, dynamic early exit, KV-cache recycling, adaptive lookahead — but exposes them as composable Python primitives you can plug into any model interface in a few lines.

The codebase is a single Python package, NumPy-only at its core. HuggingFace, FastAPI, and every other heavyweight is an optional extra. Tests run offline against deterministic mock models so you can develop optimization passes without burning a GPU hour.

primitives

Six pieces. Compose them however you want.

Every primitive implements a clear contract, exposes its stats, and has 100% test coverage. No magic, no surprises.

SpeculativeDecoder

Draft model proposes γ tokens, target verifies in parallel, residual sampling on rejection. Implements Chen et al. 2023 / Leviathan et al. 2023 to the letter.

kairu.SpeculativeDecoder

LayerwiseEarlyExit

Walks transformer layers and emits the argmax token at the first layer whose top-prob clears the threshold. Skip up to 60% of layer compute on confident tokens.

kairu.LayerwiseEarlyExitDecoder

LogitsCache

Bounded LRU cache memoizing next_token_logits(prefix). O(1) get/put via OrderedDict.move_to_end. Recycles target calls across speculative verification rounds.

kairu.LogitsCache · CachedModel

DynamicGammaScheduler

AIMD over speculative γ. Grows lookahead under high rolling acceptance, multiplicatively shrinks under low. Same control law as TCP. Configurable bounds, thresholds, window.

kairu.DynamicGammaScheduler

AutoProfile

Picks vanilla / early_exit / layered_early_exit / speculative from model metadata. Returns a frozen profile with explicit rationale.

kairu.AutoProfile

SSE Streaming Server

FastAPI POST /generate emitting OpenAI-compatible chat.completion.chunk frames with per-token kairu metrics. Boundary-validated, rate-limited, timeout-enforced.

kairu.create_app

Layerwise early exit, in motion

Each token enters the transformer at layer 1 and rises until its top-prob crosses the threshold. Skipped layers are wasted compute kairu doesn't pay.

How it reads

The 12 horizontal lanes are transformer layers (bottom → top). Each rising column is a token; it stops at its exit layer. Confident tokens exit early. Hard tokens climb further.

The dashed band is the confidence threshold. Below it, the model isn't sure enough yet — it keeps computing. Above it, kairu emits the argmax and moves on.

Five lines to wrap a model.

Every primitive is independently composable. Use the wrapper for a fully wired stack, or reach for a single decoder when you need it.

# Auto-pick strategy + cache size for any model
from kairu import AutoProfile, ModelWrapper

profile = AutoProfile.recommend(model,
    name_hint="llama-3-8b",
    has_draft=True)

print(profile.strategy)   # speculative
print(profile.gamma)      # 4
print(profile.rationale)  # "large target..."

wrapper = ModelWrapper(model,
    draft_model=draft,
    cache_capacity=256,
    adaptive_gamma=True)

# Stream OpenAI-compatible SSE chunks
from kairu import create_app, ServerConfig
import uvicorn

app = create_app(config=ServerConfig(
    model_name="kairu-mock",
    rate_limit_requests=10,
    request_timeout_s=30.0,
))
uvicorn.run(app, host="0.0.0.0", port=8000)
